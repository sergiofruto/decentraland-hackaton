Whats new?

Introducing the SDK 6.3

The Decentraland SDK is based on TypeScript (JavaScript with type checking), featuring development patterns that are common among game development platforms. The upcoming 6.3 version will support collision events, moving players around, a polished HUD interface, and more.

An integrated art pipeline

SDK + Builder

We recommend using the Builder, Decentraland's visual editor, as a starting point for your art pipeline. Import and position pre-made, pre-loaded custom 3D models. Export the scene to dive into the code and make it fun!
